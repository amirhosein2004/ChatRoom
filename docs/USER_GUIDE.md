# User Guide

Quick guide to using the chat application.

## Accessing the Application

### Room List Page
```
http://localhost:8000/api/room/v1/rooms/list/
```
Browse and create chat rooms.

### Public Chat
```
http://localhost:8000/api/room/v1/chat/
```
Public chat room for everyone.

### Specific Room Chat
```
http://localhost:8000/api/room/v1/chat/{room-slug}/
```
Chat in a specific room.

---

## Creating a Room

1. Go to room list page
2. Click "ایجاد اتاق جدید" (Create New Room)
3. Enter room name and description
4. Click "ایجاد اتاق" (Create Room)

---

## Joining a Room

1. Browse rooms on room list page
2. Click "ورود به اتاق" (Enter Room)
3. Enter your username (first time only)
4. Start chatting!
